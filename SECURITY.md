# Security Policy

Please report vulnerabilities privately through this repository's
[GitHub security advisory form](https://github.com/estridell/oh-my-grok/security/advisories/new).
Do not open a public issue for a vulnerability.

This policy applies only to oh-my-grok. Vulnerabilities in upstream Grok Build
or xAI services should be reported through the security channels published by
their respective maintainers.
